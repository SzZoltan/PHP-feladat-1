# PHP-feladat-1
Versenykezelő Webalkalmazás Laravel framework segítségével

Rövid leírás:
Készíts egy webalkalmazást, ahol versenyeket lehet felvenni, hozzájuk pedig versenyzőket
lehet csatolni.

Funkciónális elvárások:

Az oldalon alaphelyzetben megjelenik az Új verseny felvétele gomb, amivel egy versenyt
felvehetünk az adatbázisba. Ezt követően a már létező versenyekhez felvehetünk fordulókat,
a felvett fordulókhoz hozzárendelhetünk versenyzőket. A versenyzőket is az admin tudja
felvenni, erre kialakított felületen. Egy felhasználó több verseny, több fordulójához is
hozzárendelhető.
Egy verseny a neve és verseny éve alapján egyedi, de ezeken kívül legyen még több
tulajdonsága is! (pl. elérhető nyelvek, pontok jó, rossz, üres válaszért stb.)
Egy felhasználó az email címe alapján egyedi, de ezeken kívül legyen még több
tulajdonsága is! (pl. telefonszám, lakcím stb.)

Alkalmazott technológiák:
● GitLab vagy GitHub

● UTF-8 kódolás

● Mobilbarát, reszponzív változat
